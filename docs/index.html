Now available on Chrome Web Store

Hold up.

A personal nudge engine for your browser. You set the reminders. You write the messages. Holdup makes sure you read them.

Install from Chrome Web Store View on GitHub

How it works

1

Add a domain + your message

Pick overlay or fullscreen mode. Choose your cooldown.

2

Visit that site

Holdup intercepts the navigation.

3

Read your own words. Decide.

Go back, continue, get reminded later, or redirect somewhere better.

No data collection. No servers. No analytics. Everything stays in your browser.
