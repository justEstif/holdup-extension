# DESIGN.md — Holdup

## Visual identity

Calm authority. Warm slate + amber. System fonts. Generous space. Nudge = pause, not panic.

## Quick reference

- **Colors**: slate (neutral), amber (accent/warmth), red (destructive only)
- **Typography**: system font stack, weight contrast (bold headings, regular body)
- **Spacing**: generous — nudges breathe, 48px card padding on interstitial
- **Motion**: minimal, purposeful. Overlay fade-in only. Interstitial instant.
- **CTA logic**: "Go back" = amber (warm, encouraged). "Continue" = slate (muted, available but not promoted)
- **No-go**: no blue primary, no red on nudges, no web fonts, no custom CSS files

## Surface hierarchy

1. **Interstitial** — dark bg, full viewport, maximum visual weight
2. **Overlay** — light card on frosted backdrop, strong but respectful
3. **Options** — clean tool UI, functional over expressive
4. **Popup** — minimal, status only
